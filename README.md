# taskman-frontend

## Project setup
```
npm install
```

Create new `.env` file then copy the values below:

```
NODE_ENV=development
VUE_APP_API_URL=http://localhost
```

Compile and hot-reloads for development:

```
npm run serve
```

Goto `http://localhost:8080/ ` then login using the ff creds:
```
email: jdc@test.com
password: password
```
